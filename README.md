# Movie Review App 
A React app that displays popular movies using the **TMDB API**. Users can browse movies and view view details.
Built with **Vite**, **React**, and **CSS**, fully responsive and fast.

## overview
The Movie Review App is a React-based web application that allows users to browse movies and view detailed information along with reviews and ratings. The application provides a dynamic and interactive user experience with fast navigation and real-time UI updates.
This project demonstrates the use of React components, state management, props handling, and API integration to fetch movie data. It is designed to simulate a real-world movie browsing platform where users can explore trending movies and read reviews in a clean and responsive interface.

## Features
- Browse popular movies from TMDB  
- View movie details including ratings and overview  
- Search for movies by title  
- Responsive design for desktop & mobile  

## Tech Stack
- **React** – Frontend library  
- **Vite** – Development server & build tool  
- **TMDB API** – Movie data  
- **CSS** – Styling  
- **ESLint** – Code quality
