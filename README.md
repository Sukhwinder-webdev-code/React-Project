# Movie Review App 
A React app that displays popular movies using the **TMDB API**. Users can browse movies and view view details.
Built with **Vite**, **React**, and **CSS**, fully responsive and fast.  

---

## Live Demo
ViewLive:https://github.com/Sukhwinder-webdev-code/React-Project

---

## Features
- Browse popular movies from TMDB  
- View movie details including ratings and overview  
- Search for movies by title  
- Responsive design for desktop & mobile  

---

## Tech Stack
- **React** – Frontend library  
- **Vite** – Development server & build tool  
- **TMDB API** – Movie data  
- **CSS** – Styling  
- **ESLint** – Code quality
